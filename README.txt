HOW TO USE STUDY_TRACKER.PY

*Requires Python 3.4 & Above.
*study_tracker.py is currently configured to run in Terminal.
*It is highly recommended that you store this program in an easily accessible folder.
*A log.json file will be created in the same directory. This file contains your saved study log.

1. Open the folder that study_tracker.py is located in
2. Right click empty space
3. Click "Open in Terminal"
4. Type "python study_tracker.py"
5. Type Enter

Please report any bugs. More features coming soon.
